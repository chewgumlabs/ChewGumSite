# Site Notes

- Treat this folder as the document root when serving the site.
- The current canonical domain assumption is `https://shanecurry.com`.
- The GitHub Pages workflow publishes this `site/` folder directly.
- Until the custom domain is active, the temporary `github.io` preview will not fully match the final canonical URL setup.
- Add new thought chunks under `blog/<slug>/` first, then cross-link or promote them into topic pages when useful.
- When a new page is added, update `blog/index.html`, `animation/index.html`, `sitemap.xml`, `feed.xml`, and `llms.txt`.
