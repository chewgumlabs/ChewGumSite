Build Notes

The tunnel is not a square room. It is a long faceted tube whose cross section shifts ring by ring, so it reads more like an icosphere stretched into an endless hallway.

The view is rendered into a low-resolution framebuffer and scaled back up with crisp pixels and scanlines, so it stays close to the site’s existing software rendering experiments.

Bell Logic

Each wall strike triggers a small bell built from the same triangle-wave generator pattern introduced on Dead Beat.

This piece treats falling like navigation. You are always descending, but the tunnel pulls and twists just enough to make the drop feel physical. The center stays dark and distant while the walls pick up ring seams, facet edges, and impact flashes as they rush past.
