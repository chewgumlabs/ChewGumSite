This index collects small public software extractions that have a real repository, a bounded API, and a source trail back to shipped work on this site.

  • ChewGumDitheringGun v0.0.1: a small browser dithering microscope for dropping images under resolution, palette, and dithering constraints.
  • ChewGumTimeChime v0.1.0: a small browser harness for comparing stroke smoothing algorithms through timed chimes.
  • ChewGumDSP v0.1.0: a narrow Web Audio extraction for NES-style triangle waves, MIDI frequency conversion, and bell synthesis.

A tool belongs here when the public repository exists, the version is named, and the page can connect the API to deployed artifacts without exposing private research code.
